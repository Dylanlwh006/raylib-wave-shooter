# Raylib Wave Shooter

A C++ Raylib top-down wave shooter final project.

## Features
- Player movement
- Gun follows mouse cursor
- Shooting system
- Enemy waves
- Collision detection
- Lives and money system
- Upgrade shop
- Save/load system
- High score system

## Technologies
- C++
- Raylib

## Project Structure
- `src/` - source code
- `assets/` - images, sounds, fonts
- `data/` - save files and high scores
- `docs/` - project documents
